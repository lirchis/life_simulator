# 早年死亡的年代校准

事件库中的死亡风险服务于文学模拟，不是精确人口生命表。但如果晚清、民国和当代人物拥有几乎相同的幼年存活率，完整人生会在最基本的时代事实上失真。

## 证据基线

- Babiarz 等基于历史公共卫生档案、省级年鉴和生育调查重建 1950—1980 年省级数据，概括 1949 年中国出生预期寿命约 35—40 岁，1980 年升至 65.5 岁，并指出教育和公共卫生干预对婴儿、五岁以下死亡下降有重要贡献：<https://pmc.ncbi.nlm.nih.gov/articles/PMC4331212/>。
- 2026 年的省级纵向研究估计，1950 年各省婴儿死亡率约为每千活产 120.0—368.4，省级均值 217.3；地区差异非常大：<https://pmc.ncbi.nlm.nih.gov/articles/PMC13158603/>。
- 基于 GBD 的研究给出 1990 年全国婴儿死亡率约每千活产 43.2，2019 年约 6.8：<https://pmc.ncbi.nlm.nih.gov/articles/PMC8393072/>。

这些数字不是拿来把每一局强行配平。样本小、早期登记不完整、地区和阶层差异大，真实资料本身也有不确定性。

## 当前文学模拟口径

引擎按年代使用独立事件，避免现代儿童继承近代风险。基础概率还会被低健康、低体质和低医疗可及性放大。

| 年代 | 新生儿基础概率 | 1—5 岁每年急病概率 |
| --- | ---: | ---: |
| 1840—1949 | 9.0% | 2.5% |
| 1950—1977 | 5.0% | 1.2% |
| 1978—1999 | 1.8% | 0.4% |
| 2000—2035 | 0.6% | 0.08% |

这里有意低于部分历史人口估计：玩家按“下一年”阅读一部长篇人生，原样套入高死亡率会让大量作品只剩一两段。当前目标是恢复清楚的时代梯度、阶层梯度和医疗梯度，同时保留足够多可供阅读的完整人生。

早期版本只有 0—5 岁夭折和 75 岁后的自然死亡，批测会形成 6—74 岁几乎无人去世的假空洞。现在另设低概率的 6—14、15—39、40—64、65—74 岁重病结局，并按四个历史阶段分别校准；低健康、低体质、低医疗可及性和矿井、码头、建筑、工厂等高风险劳动会放大风险，健康状况良好则降低风险。它们只补回现实中确实存在的中途结束，不把战争、贫困或疾病当作戏剧奖品。

内容评测不再用跨年代统一的“40 岁前死亡不得超过 15%”惩罚线。晚清、民国、建国初期、改革年代和当代分别使用 45%、40%、25%、10%、5% 的异常上限，并要求该年代至少有 200 局才据此扣分；这些是发现程序性尖峰的宽松护栏，不是人口学目标值。小样本只展示分布、进入复核，不用一次 seed 波动代替结论。下限不足仍会进入分层审查，避免为了让平均寿命好看而删去历史死亡。

## 评测要求

- 分层批测必须分别报告各出生世代的婴儿、18 岁前、40 岁前结束率与终年分布。
- 必须检查死亡年龄是否出现长段空洞；不能让 6—74 岁人物获得无依据的事实永生。
- 不能只看所有人生的平均终年；婴幼儿早亡与成年幸存者的寿命必须分开人工阅读。
- 早亡文案采用家人、照料和物件的外部镜头，不给婴幼儿安上稳定的自传式记忆。
- 任何概率调整都要同时检查不同地区、城乡、性别、体质和医疗可及性，避免把全国均值写成每个人的命运。
