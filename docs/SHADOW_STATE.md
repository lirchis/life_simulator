# 隐藏阴影叙事状态

`state.shadow` 用来记录一生中不容易由健康、财富、成就和幸福表达的后果：防御性变硬、责任纠缠、已经造成的伤害、自我辩解、积怨与被透支的信任。

它是叙事连续性状态，不是善恶分、人格诊断、司法裁决或“恶人值”。高值不自动意味着一个人更坏，低值也不证明清白。现实允许伤害者没有受到惩罚，也允许受害者产生怨恨；评测要求的是文本不要把已经发生的东西凭空擦掉。

## 字段

所有字段都是 `0—100` 的隐藏整数：

| 字段 | 叙事含义 | 注意 |
| --- | --- | --- |
| `hardness` | 为承受压力或反复越过他人感受而形成的硬壳 | 既可能来自受苦，也可能伴随施害，不等于恶 |
| `guilt` | 人物实际感到的愧疚和责任压力 | 没有愧疚不等于没有造成伤害 |
| `complicity` | 对组织、制度或群体行动的卷入程度 | 不直接断言主观恶意 |
| `harmDone` | 一生已经造成伤害的累计叙事痕迹 | 原则上不因道歉或未受惩罚而减少 |
| `selfDeception` | 为维持自我形象而形成的辩解、回避和改写 | 可以与真诚、善意或愧疚同时存在 |
| `resentment` | 长期受损、受限或关系失衡留下的积怨 | 不是对受害者的道德扣分 |
| `trustDebt` | 对具体关系造成的信任透支 | 修复需要坦白、赔偿、持续行动等叙事依据 |

`shadow.threads` 另按结构链保存局部状态。目前包含 `guilt`、`justification`、`benefitRetained`、`responsibilityAccepted`、`victimContact`。它解决的是“串戏”：一个人可以因伤害孩子而愧疚，却仍在另一条职场甩锅线上坚信自己无错。文案判断某一旧事是否悔悟、辩解或保留收益，应优先读取本线程；全局七字段只描述整个人生累计形状。

## 初始化与自动演化

- `guilt`、`complicity`、`harmDone`、`trustDebt` 一律从零开始，不从家庭阶层、户口、地区或财富推断。
- 心态与家庭物质不安全感只会非常克制地影响低个位数的 `hardness`、`resentment`。智力不参与道德或诚实推断，所有人物以相同的少量自我保护性解释倾向开局；低资源环境最多增加一点防御状态，不被解释为更不道德。
- 严重的自由、幸福或关系损失可缓慢增加硬化和积怨；“获得地位或资源，同时关系/自由显著受损”的事件只会轻微增加卷入和自我辩解。
- 年度自然演化幅度很小。累计伤害、愧疚和信任债不会由贫穷、疾病或坏运气自动生成。
- 新阴影线程只依据人物当时的防御状态初始化少量“愿意正视/倾向辩解”的压力，后续由本链事件推进；别的链产生的 `guilt` 不再直接解锁本链道歉文案。私人链结尾也不会无条件增加愧疚。
- 事件需要明确表达责任时，应像其他状态一样使用 effect：

```js
effects: [
  { path: "shadow.harmDone", add: 8 },
  { path: "shadow.trustDebt", add: 6 },
  { path: "shadow.guilt", add: 3 },
]
```

事件条件可直接读取：

```js
conditions: {
  all: [
    { path: "shadow.harmDone", gte: 8 },
    { path: "shadow.selfDeception", gte: 10 },
  ],
}
```

跨年后果如果正文写“后来”“渐渐”或“多年后”，应同时声明最短和最长间隔，避免三段人生在连续两年内匆忙走完：

```js
{ eventOccurredBetween: { eventId: "shadow_example_open", minYears: 2, maxYears: 8 } }
```

道歉、赔偿或长期修复可以降低 `trustDebt`，但不应把累计 `harmDone` 清零。人物也可以不悔改、逃脱处罚或继续自我辩解；这不是引擎错误。

## CSV 与专项审查

普通批测和分层批测均导出：

- summary：`shadow_*_final`。
- years：`shadow_*_before`、`shadow_*_after`，分别表示年初/上年末与本年结束状态。
- events：精确的单事件 `shadow_*_before`、`shadow_*_after`；同年多事件不会共用一个模糊快照。

`review_stratified.js` 输出以下指标和人工审查告警：

- 阴影状态发生变化的事件数、人生数，以及七字段的终局均值、P90、最大观察值。
- 主动伤害叙述后十五年内是否留下责任、关系、记忆、自我辩解等痕迹；人物在窗口内死亡时不误报。
- 是否只在结尾安排一次悔悟，替代中段持续的后果。
- 单事件超过 25 点的跳变。
- `harmDone` 倒退、信任债无修复依据地大幅消失、同年相邻事件状态断裂等矛盾。
- 同一人生是否塞入过多阴影链，以及性别、阶层、城乡的触发率是否出现无法由处境解释的显著失衡。

“没有受到法律、财富、名声或死亡上的惩罚”不会单独触发错误。告警检查的是叙事是否失忆，不负责替现实执行报应。
