# 人生模拟器

纯前端人生模拟器原型。

## 启动

```bash
npm run start
```

然后打开：

```text
http://localhost:4175
```

也可以使用任意静态服务器打开仓库根目录。由于前端使用 ES Modules，不建议直接用 `file://` 打开 `index.html`。

## 当前实现

- 壳子 / 数据分离
- 固定 seed 随机
- 初始状态手选和一键随机
- 年度推进
- 连续时间轴展示
- 条件系统
- 权重系统
- 效果系统
- 过去状态快照
- 事件依赖
- 跨时间 modifier
- 地区聚合
- 选择事件
- 死亡和人生报告
